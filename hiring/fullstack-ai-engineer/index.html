Responsibilities

Two hats. One engineer.

AI Agent Platform

  • Own and maintain the AI agent platform — already in MVP, shipping to production
  • Build new agent features end-to-end — React frontends, Node.js/TypeScript backends, GraphQL APIs
  • Interface with other IFCA product APIs (FMX, RentX, HotelX, AuditX, IFCAx) to extend agent capabilities
  • Integrate open-weight LLMs (Minimax, Llama, Qwen) and commercial APIs (Anthropic, Google Gemini)
  • Implement safety guardrails — output filtering, hallucination checks, permission boundaries, rate limits

GPU Infrastructure

  • Coordinate GPU deployments with hardware vendors — cloud, on-prem, and hybrid
  • Work alongside vendor engineers for NVIDIA GPU server setup (H100, H200, RTX 6000 Ada) at client sites
  • Configure inference engines — vLLM, TGI, or TensorRT-LLM
  • Monitor performance, optimise throughput, manage model serving at scale

Three deployment models.
