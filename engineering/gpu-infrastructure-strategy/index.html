Watch Out

Three things that can go wrong.

Price Crash

H100 prices are still falling. Don't stockpile hardware. Buy per-deal or use GPUaaS until margins stabilise.

Security Liability

Deploying at client DCs means shared responsibility. Insist distributor provides ISO 27001 hardening docs and sign-off.

Talent Gap

Need to hire GPU infra engineers or rely entirely on distributor. Hybrid model recommended — IFCA owns software, distributor owns hardware ops.
