Best Practices

How to get the most out of AI agents

CLAUDE.md

Write a project instructions file. Teach the agent your conventions, folder structure, and rules. It reads this every session.

Be specific

Clear, detailed prompts beat vague instructions. Include file paths, function names, expected behavior. The agent is literal.

Review, don't rewrite

Let the agent generate, then review and iterate. Your role shifts from writer to editor and architect.

Use Opus

Model quality matters. Opus one-shots complex tasks that Sonnet or GPT struggle with. Pay for the best model — it saves time.
