Decision Framework

Declare PMF when…

  • Sean Ellis >40% "very disappointed"
  • Logo retention >95%
  • NRR >115%
  • Sales cycle <150 days
  • Dashboard usage >70%
  • Unprompted referrals

If PMF not achieved…

  • Wrong segment → pivot focus
  • Product gaps → prioritise features
  • Pricing mismatch → adjust model
  • GTM issue → change sales motion
  • E-Invoice issues → immediate fix
